**bytebeat workstation** 🎛️  
A web-based bytebeat synthesizer and visualizer with dynamic theming.

**Try it live here:** https://nemusya.github.io/bytebeat.workstation

**Features**
- 🎵 Real-time formula evaluation and audio generation
- 🎨 6 smooth theme presets with live waveform updates
- 🔊 Volume and sample rate controls
- 💾 30-second WAV export
- 📜 9 built-in example formulas (neurofunk, arcade, glitch, etc.)
- 🔒 Hardened with CSP headers, X-Frame-Options DENY, nosniff, strict referrer policy

Built with vanilla JS, CSS variables, and the Web Audio API. Shoutout to https://chat.deepseek.com for the code companionship!
